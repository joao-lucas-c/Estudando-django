Projeto de Estudo Django 

Este repositório contém meus primeiros passos com o framework Django. 

O que pratiquei aqui:
* Configuração de Ambiente Virtual (`venv`).
* Estruturação de Apps e rotas (`urls.py`).
* Gerenciamento de arquivos estáticos (CSS e Imagens).
* Resolução de conflitos de nomes de módulos.
